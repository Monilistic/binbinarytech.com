Software development is the process of designing, programming, documenting and mainting applications. Software development involves writing and maintaning the source code, it includes research, new development, prototyping or any other activities that result in software products

A software plays a key role of a mediator between the user and the computer hardware. In the absence of software, a user can't perform any task on a computer. A software product development company is the one which develops software for the users.

There are basically two main classification of software, which are namely, 1. System Software and 2. Application Software.

System Software helps the user as well as the hardware to function and even interact with each other easily. In order word, System Software is essentially an intermediator between the user as well as the hardware.
